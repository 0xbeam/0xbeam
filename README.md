## The Stack

All UI projects share the **Leonardo Design System** — editorial minimal aesthetic, forest green accent (`#2A7A5B`), warm neutrals, Cormorant Garamond + DM Sans typography. Tokens live in [`leonardo`](https://github.com/0xbeam/leonardo), get pushed to every app via `node build.js --push`.
